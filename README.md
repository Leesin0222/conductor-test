# Perth 🐱

macOS 메뉴바에 사는 보안 펫 앱. 클립보드를 실시간으로 감시하여 민감한 개인정보가 감지되면 펫이 경고해줍니다.

## 기능

### 펫 & UI
- **4종 펫 캐릭터**: 고양이, 강아지, 토끼, 곰 중 선택
- **시간대별 상태**: 오전 6시~오후 11시는 활성(happy), 그 외 시간은 취침(sleeping) 상태
- **위험 감지 시 상태 변화**: 감지 직후 경계(alert) 상태로 전환, 15초 후 자동 복귀
- **온보딩 플로우**: 첫 실행 시 3단계 안내 (환영 → 펫 선택 → 기능 소개)

### 클립보드 감시
- 설정 가능한 감시 주기 (기본 3초)
- 감지 패턴별 개별 활성화/비활성화
- **자동 클리어**: 민감한 정보 감지 후 설정된 시간(기본 5초) 경과 시 클립보드 자동 초기화
- **앱 제외 목록**: 특정 앱(기본값: 1Password, KeePassXC)에서 복사한 내용은 감시 제외

### 알림 탭
- 최근 감지된 알림 목록 표시
- 알림 일괄 지우기
- 클립보드 즉시 지우기 버튼

### 히스토리 탭
- 최근 100건의 클립보드 복사 이벤트 기록
- 복사 시각, 소스 앱 이름, 감지된 패턴 유형 표시

### 통계 탭
- 오늘 / 이번 주 감지 횟수
- 최근 30일 일별 통계
- 이번 주 가장 많이 감지된 패턴 유형
- **안전 연속 일수 (Safety Streak)**: 마지막 감지 이후 경과 일수 표시
- JSON 형식으로 통계 내보내기

### 패턴 탭
- 정규식 기반 커스텀 패턴 추가/삭제/활성화 토글
- 패턴 유효성 실시간 검증

### 설정 탭
- 감시 켜기/끄기
- 감시 주기 조절
- 알림 사운드 모드: 기본 / 위험도별 / 무음
- 자동 클리어 활성화 및 지연 시간 설정
- 앱 제외 목록 관리
- 로그인 시 자동 실행 (Launch at Login)
- 펫 캐릭터 변경

### 화면 공유 감지
- Zoom, Teams, Discord, Webex, macOS 기본 화면 공유 실행 중 배너 경고 표시
- 5초 간격으로 상태 감지

## 빌드 & 실행

```bash
# 빌드
swift build

# 직접 실행
swift run Perth

# .app 번들 생성
make bundle
open .build/Perth.app
```
